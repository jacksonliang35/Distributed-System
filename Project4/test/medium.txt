ISO-Aka-Titles List                                                1 March 1996
===============================================================================

Document Revision 1.0 - 6 September 1995

   This alternative title-list offers you foreign movie titles with ISO
   Latin 1 characters. In the future maybe all software will support
   international characters. Until then, this file can be used optionally
   by those who can use ISO Latin 1 characters. This list is compatible
   with the other aka lists used in the internet movie database.

   If you have any additions or corrections please study the sections 4 and
   5 of this list.

   IMPORTANT NOTE on international characters: Starting with this list we
   support international characters with the ISO Latin 1 character set. If
   you do not want or are not capable of using ISO Latin 1 characters,
   please do not use this list. Many Unix platforms, MS Windows and the
   Amiga support ISO Latin 1 fully by default.
   Please check out, if your system can display these characters:
   A umlaut        = ä
   E accent grave  = ê
   UK Pound sign   = £


   Enjoy!
